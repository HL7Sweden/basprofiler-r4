ValueSet: SE ValueSet for marital status

Official URL: http://hl7.se/fhir/ig/base/ValueSet/SEBaseMaritalStatusVS Version: 1.1.0
Active as of 2025-05-20 Computable Name: SEBaseMaritalStatusVS

This is a value set of marital status types that include both the HL7 official codesystem and the ones specific from the Swedish authorities.
