Data Type Profile: Profile for LMA number

Official URL: http://hl7.se/fhir/ig/base/StructureDefinition/SEBaseLMANummer Version: 1.1.0
Active as of 2025-05-20 Computable Name: SEBaseLMANummer

This profile is to be used specifically for expressing an LMA number as an identifier

Usage:

  • This DataType Constraint is not used by any profiles in this Implementation Guide
