Data Type Profile: SEBaseHumanName

Official URL: http://hl7.se/fhir/ig/base/StructureDefinition/SEBaseHumanName Version: 1.1.0
Active as of 2025-05-20 Computable Name: SEBaseHumanName

This profile provides the use of Swedish middlename and own family name as extensions and is used in the base Patient profile

Usage:

  • Use this DataType Constraint: SE base profil for patient and SE base profile for practitioner
